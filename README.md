## Quick Commands (cheat sheet)
**Navigation:** `pwd`, `ls -la`, `cd`, `find`, `cat`, `less`  
**Permissions:** `chmod`, `chown`, `umask`  
**Processes:** `ps aux`, `top`, `kill`, `nice`  
**Services/Logs:** `systemctl status`, `journalctl -xe`, `tail -n 100`, `grep`  
**Disk/Memory:** `df -h`, `du -sh`, `free -m`  
**Network:** `ip a`, `ping`, `curl`, `ss -tulpn`
